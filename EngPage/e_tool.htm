Puzzle007 Copy     Copy multi-files with speed limit. Based on Nullsoft Copy v0.1.
Puzzle007 FileNameChecker     Microsoft has been using Unicode for filenames for many years, but unfortunately, many software still handle filename in multibyte. If a file has a name containing characters not in windows' current codepage, it can't be handled by these old-style softwares. For example, Nero5 refuses to add some Japanese filename in my Chinese-Windows. Nero6 is better, but these file names will be converted to 8.3 format.
    Puzzle007 FilenameChecker helps handle this situation. After specified a folder to check, it will find out the filenames having the problem described above, and prompt the user to give new name to the files.
Puzzle007 FileSizeChanger     This command line tool can change file to any size you desired. Zeros are appended if the file is extended. If the file is truncated, all data from the end of the shortened file to the original length of the file is lost.
Puzzle007 EncodingConvertor     Convert text between Simplified Chinese and Japanese.
